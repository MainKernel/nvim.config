# Головна клавіша (Leader Key): Space (Пробіл)

## Файли та Навігація
- Space + e : Відкрити/закрити дерево файлів (Neo-tree) 
- Space + f f : Пошук файлів (Telescope) 
- Space + f g : Пошук тексту по проєкту (Live grep) 
- Space + f b : Пошук серед відкритих буферів 
- Space + f h : Пошук по документації 
- Space + f t : Вибрати і вставити шаблон

## Робота з кодом (LSP)
- g d : Перейти до визначення 
- K : Показати документацію 
- Space + r n : Перейменувати змінну/функцію всюди 
- Space + c a : Швидкі дії з кодом (Code actions)

## Автодоповнення (nvim-cmp)
- Ctrl + Space : Викликати меню підказок 
- Tab / Shift+Tab : Навігація по списку підказок 
- Enter : Підтвердити вибір 
- Ctrl + b / f : Прокрутити документацію

## Сесії (Persistence)
- Space + q s : Відновити сесію для поточної папки 
- Space + q l : Відновити останню збережену сесію 
- Space + q d : Зупинити збереження сесії при виході

## Термінал (Toggleterm)
- Ctrl + \ : Відкрити/сховати термінал 
- Esc (у терміналі) : Вийти з режиму вводу термінала 
- Ctrl + h / j / k / l : Перехід у вікна з кодом

## Інше
- Space + ? : Відкрити пам'ятку у вертикальному спліті
